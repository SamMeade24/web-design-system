Colour's

The following colours are used throughout the webpage.

White

Hex: #FFFFFF

This colour is used as my primary font colour, and occasional background colour too.

Dark Green

Hex: #145A32

This colour is used as my primary background colour, and font colour for lighter backgrounds.

Light Green

Hex: #52BE80

This colour is used as my lighter background colour option.

Gold

Hex: #F4D03F

This colour is used as the colour of my headings, and any button backgrounds.

Text Style's

The following are specififc text styles for for different page elements.

H1: Main page heading

  • Font-weight: 700 (bold)
  • Font-size: 26px
  • Font-Family: Anton

H2: Subheading

  • Font-weight: 500
  • Font-size: 18px
  • Font-Family: Roboto

P: Paragraph Text

  • Font-weight: 400 (regular)
  • Font-size: 14px
  • Font-Family: Roboto
